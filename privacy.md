# Política de Privacidade

Esta Política de Privacidade explica como os dados são usados no bot.

## Dados coletados
O bot pode armazenar:
- ID do usuário
- Dados de jogo (nível, fichas, status, etc.)

## Uso dos dados
Os dados são utilizados apenas para funcionamento do bot (RPG, sistema de jogo, etc.).

## Compartilhamento
Nenhum dado é vendido ou compartilhado com terceiros.

## Segurança
Os dados são armazenados de forma básica e podem ser apagados a qualquer momento.

## Direitos do usuário
Você pode parar de usar o bot a qualquer momento, o que interrompe a coleta de dados.

## Alterações
Esta política pode ser alterada a qualquer momento sem aviso prévio.
